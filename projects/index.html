Mask Detection Monitor

A tool that uses Convolutional Neural Networks to calculate a Mask Score in a populated area.

I collaborated with Zeyad Mansour to create a Mask Detection Monitor for the 2021 HS Hack OSU Hacakthon (1st place project). Our project uses OpenCV and Neural Networks to determine whether or not people are wearing masks. You can check out our web-app here (currently down while we figure out hosting issues) or our GitHub repository here.

Sang Bot

An auto-moderator Discord bot for MHSCTF.

For MHSCTF, I created a Discord server and a bot to help me run this server. Sang Bot uses artificial intelligence to moderate messages and helps out in regular administrative processes on a Discord server. I hope to soon be able to release Sang Bot for public use.
